---
title: "Invoice statuses"
---

List of possible statuses of the created invoice.

| Numerical identifier | Description                                          | Final     |
|:-----:|---------------------------------------------------------------------|:---------:|
|  1    | Payment request has been created                                    | &minus;   |
|  2    | Payment is awaiting user action                                     | &minus;   |
|  3    | Payment system has failed to create the payment                     | &minus;   |
|  4    | Payment has been successful                                         | &plus;    |
|  5    | Payment system has failed to receive a payment notice               | &minus;   |
|  6    | M4  payment system has rejected the operation               | &plus;    |
|  8    | Funds for the invoice are withheld (held) by the payment system     | &minus;   |
